# HavocPvP - Système de Gamemode
Ce système appartient désormais à HavocPvP.

## Structure
- `gamemode-manager/` - Gestionnaire principal des modes de jeu
- `lobby-system/` - Système de lobby avec interface NUI
- `spawn-manager/` - Gestionnaire de spawn amélioré
- `maps/` - Maps séparées du système de gamemode
- `gamemodes/` - Modes de jeu individuels

## Gamemodes Inclus
- **havoc-freeroam** - Mode libre avec spawn automatique
- **havoc-deathmatch** - Mode combat PvP compétitif avec statistiques

## Maps Incluses
- **havoc-map-city** - Points de spawn urbains (Los Santos)
- **havoc-map-arena** - Arène de combat (Fort Zancudo)

## Fonctionnalités
- Système de lobby avec sélection de mode de jeu
- Gestion séparée des maps et gamemodes
- Interface NUI moderne avec design HavocPvP
- Gestion des joueurs (spawn, vie, armure)
- Statistiques de jeu en temps réel
- Support multi-gamemode automatique
